Arjan Audit

Audit your site pages with Lighthouse 6 in development mode and save the results locally. Await future feature releases to unlock the power of your data!

Get Started Docs

Test your site's speed and SEO score locally before deploying

How it Works

Arjan audit uses express to serve your site and Chrome-launcher to start Google Chrome in headless mode and retrieve the audits from Lighthouse 6.

Audit Report

Arjan audit parses the results returned from lighthouse 6 and returns an organized subset with recommendations based on your score and threshold. All the audits are saved locally.

Get Started

  1. If you haven't already, install Arjan globally
  2. Go into your project's directory and initialize Arjan (only do this once per project =)
  3. Run an audit
  4. For more info you can always run or checkout the docs
